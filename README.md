# Профориентационные тесты (статические)

Коллекция статических офлайн-тестов, которые работают целиком в браузере без бекенда и сторонних библиотек. Главное меню расположено в корне, каждый тест живёт автономно внутри своей папки.

## Структура
- `index.html` — главное меню со списком доступных тестов.
- `tests/neurocareer/` — готовый тест «Ваш мозг в IT».
  - `index.html` — каркас приложения и экраны (старт, вопросы, результат).
  - `style.css` — тёмная адаптивная тема, стили карточек, кнопок и прогресса.
  - `test-data.js` — все вопросы, варианты ответов и описания профилей.
  - `app.js` — логика показа вопросов, навигации, подсчёта результатов и экспорта.
  - `img/` — локальные иллюстрации профилей.
- `tests/cognitive/` — тест «Когнитивный профиль» (25 заданий, таймер 20 минут).
  - `index.html` — интерфейс старта, прохождения и результата.
  - `style.css` — оформление таймера, прогресса и списков результатов.
  - `test-data.js` — 25 заданий (choice/number) и шкалы интерпретации.
  - `app.js` — таймер, подсчёт баллов, управление пропусками и экспорт JSON.
- `tests/test2/` — заглушка для второго теста (ожидает ТЗ).

## Запуск
1. Склонируйте репозиторий.
2. Откройте `index.html` в браузере (двойной клик или `python -m http.server 8000` и перейдите на `http://localhost:8000`).
3. Выберите тест в меню. При необходимости можно открыть конкретный тест напрямую, например `tests/neurocareer/index.html`.
4. Для GitHub Pages — опубликуйте репозиторий, ссылка будет вида `https://<username>.github.io/neurocareer-web-test/`.

## Возможности теста «Ваш мозг в IT»
- 24 вопроса с вариантами А/Б/В/Г и прогресс-баром.
- Подсчёт доминирующей буквы, тай-брейк по 1-му и 4-му вопросам.
- Экран результата с описанием профиля и экспортом в JSON.
- Адаптивный интерфейс и плавные анимации между вопросами.

## Экспорт
- Кнопка «Экспорт в JSON» сохраняет ответы и итоговый профиль в файл `neurocareer-result.json`.
