# Ваш мозг в IT — статический профориентационный тест

Интерактивный офлайн-тест из 24 вопросов, который помогает определить роль в команде нейротехнологов. Работает целиком в браузере без бекенда и сторонних библиотек.

## Структура
- `index.html` — каркас приложения и экраны (старт, вопросы, результат).
- `style.css` — тёмная адаптивная тема, стили карточек, кнопок и прогресса.
- `test-data.js` — все вопросы, варианты ответов и описания профилей.
- `app.js` — логика показа вопросов, навигации, подсчёта результатов и экспорта.
- `img/` — локальные иллюстрации профилей.

## Запуск
1. Склонируйте репозиторий.
2. Откройте `index.html` в браузере (двойной клик или `python -m http.server 8000` и перейдите на `http://localhost:8000`).
3. Для GitHub Pages — опубликуйте репозиторий, ссылка будет вида `https://<username>.github.io/neurocareer-web-test/`.

## Возможности
- 24 вопроса с вариантами А/Б/В/Г и прогресс-баром.
- Подсчёт доминирующей буквы, тай-брейк по 1-му и 4-му вопросам.
- Экран результата с описанием профиля, экспортом в JSON и загрузкой PDF-сертификата (jsPDF через CDN).
- Адаптивный интерфейс и плавные анимации между вопросами.

## Экспорт и сертификат
- Кнопка «Экспорт в JSON» сохраняет ответы и итоговый профиль в файл `neurocareer-result.json`.
- Кнопка «Скачать сертификат» создаёт PDF-сертификат формата A4 с именем пользователя (или «Участник»), названием профиля, датой и уникальным номером. Сертификат генерируется целиком в браузере без сервера.
